## Objetivos del Proyecto

El objetivo principal del proyecto es una aplicación que permita ver información de todos los paises del mundo y además poder crear actividades turísticas. El proyecto cuenta con filtrados, ordenamientos y barra de busqueda.

## Funcionalidades

- Ver todos los países del mundo con paginado.
- Ver información extra de los paises como población, capital, area y subregión.
- Poder filtrar por continentes o por actividades.
- Poder ordenar alfabéticamente y población de forma descendente y ascendente.
- Poder buscar los países a través de una barra de búsqueda.
- Crear actividades turisticas y eliminarlas.

## Tecnologías usadas

- Front end: JavaScript, React, Redux, css.
- Back end: NodeJs, Express, Sequelize, PostgreSQL.
